# Write character science-reaction lines

## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
When a player annotates a notable science target (e.g. flags a transit signal on a TESS lightcurve, marks an anomaly on Mars, or logs a weather anomaly on InSight), the character could react with a short in-universe line. Design the trigger conditions and write 1–2 reaction lines per character per mission type. These would render as a brief narrative beat after game submission, before the next beat.
