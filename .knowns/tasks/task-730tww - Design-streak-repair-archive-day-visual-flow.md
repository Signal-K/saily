# Design streak-repair / archive day visual flow

## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Players can repair broken streaks by accessing archive days. This is a meaningful moment — a second chance — but currently has no celebratory or explanatory visual design. Spec the flow: what appears when entering an archive day (explanation/context), what plays when the repair is successful, and how it differs from a normal mission-complete. The archive access and streak-repair mechanics are already implemented; this unblocks a visual polish pass.
